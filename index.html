Installation

En 3 étapes

01
Installer Tampermonkey
Tampermonkey est une extension Chrome gratuite qui permet de charger GrepoPlus. Installez-la depuis le Chrome Web Store.
Installer Tampermonkey
02
Installer GrepoPlus
Cliquez sur le bouton ci-dessous. Tampermonkey s'ouvre automatiquement et vous propose d'installer le script. Cliquez Installer.
Installer GrepoPlus
03
Lancer Grepolis
Ouvrez Grepolis normalement. GrepoPlus se charge automatiquement — vous verrez les boutons du bot apparaître dans l'interface du jeu. C'est tout.
